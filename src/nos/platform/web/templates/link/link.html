Link Registry

In-memory link registry.
Links define deterministic routing logic between nodes, layers, or assistants.
Available link types are listed below. Custom links can be defined in plugins.
AlwaysLink
Always routes to the target node. Use for simple sequential workflows.
core.link
ConditionalLink
Routes based on conditional logic (if/else branching).
nos.plugins.old.if_then_workflow
LoopLink
Implements loop logic for iterative workflows.
nos.plugins.old.loop_workflow
